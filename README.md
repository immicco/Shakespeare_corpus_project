# Shakespeare_corpus_project
# Название проекта: 
Анализ корпуса сонетов Уильяма Шекспира.
# Описание: 
Корпус содержит 154 сонета Уильяма Шекспира в оригинальной английской версии с метаданными. Проект анализирует частоту и контекст упоминаний пяти человеческих ощущений (зрение, слух, обоняние, вкус, осязание), а также глаголов, относящихся к восприятию реальности, в лирике Уильяма Шекспира.
# Структура проекта: 
corpus/ - оригинальные тексты сонетов в формате .txt/
data/metadata.csv - таблица с метаданными (номер, год публикации, название цикла, если есть, указание серии)/
results/ - папка с результатами анализа (графики, таблицы)/
main.py - основной исполняемый файл проекта/
# Как запустить:

bash
python main.py
