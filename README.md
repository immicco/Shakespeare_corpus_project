# Shakespeare_corpus_project

## Название проекта:
Анализ корпуса сонетов Уильяма Шекспира.

## Описание:
Корпус содержит 154 сонета Уильяма Шекспира в оригинальной английской версии с метаданными. Проект анализирует частоту появления слова "любовь", местоимений, слов, связанных с пятью человеческими ощущениями (зрение, слух, обоняние, вкус, осязание), внутренними ощущениями, глаголов, относящихся к восприятию реальности, а также собирает данные о частеречном составе текстов, метриках лексического разнообразия и т.д. сонетов Уильяма Шекспира. На основе этого материала в проекте сравниваются группы сонетов Шекспира, а также приводится общая статистика по всему корпусу и по каждому сонету в отдельности.

## Структура проекта:
- `corpus/` - оригинальные тексты сонетов в формате .txt.
- `data/metadata.csv` - таблица с метаданными (номер, год публикации, название цикла, если есть, указание серии).
- `results/` - папка с результатами анализа (таблицы и отчёт).
    - `results/statistics.csv` - статистика для каждого сонета.
    - `results/pos_statistics.csv` - статистика распределения слов по частям речи во всём корпусе, в подкорпусах и каждом сонете.
    - `results/total_statistics.csv` - статистика для всего корпуса.
    - `results/fair_youth_statistics.csv` - статистика для цикла Fair Youth.
    - `results/dark_lady_statistics.csv` - статистика для цикла Dark Lady.
    - `results/procreation_statistics.csv` - статистика для группы сонетов Procreation.
    - `results/rival_poet_statistics.csv` - татистика для группы сонетов Rival Poet.
    - `results/report.txt` - подробный текстовый отчёт с развёрнутым описанием всех статистических данных и результатами некоторых сравнений.
- `main.py` - основной исполняемый файл проекта.
- `file_utils.py` - модуль с функциями для работы с файлами.
- `file_utils.py` - модуль с функциями для анализа текста

## Как запустить:
```bash
python main.py
```bash
